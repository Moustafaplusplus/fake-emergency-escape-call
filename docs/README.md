# Legal pages (GitHub Pages)

Static HTML for Play Store and in-app links.

| Page | File | URL after Pages is enabled |
|------|------|---------------------------|
| About | `about.html` | `https://<user>.github.io/fake-emergency-escape-call/about.html` |
| Privacy | `privacy-policy.html` | `https://<user>.github.io/fake-emergency-escape-call/privacy-policy.html` |

## Enable GitHub Pages

1. Push this repo to GitHub.
2. **Settings → Pages**
3. **Build and deployment → Source:** Deploy from a branch
4. **Branch:** `main` → folder **`/docs`** → Save
5. Wait 1–2 minutes; open the Pages URL shown in settings.

Update `LegalUrls.kt` in the app if your GitHub username or repo name differs.
